## `04` 📦 Installation

```bash
# 1. Clone the repo
git clone https://github.com/yourusername/rage-bots.git
cd rage-bots

# 2. Install dependencies
npm install

# 3. Configure environment
cp .env.example .env
# edit .env with your tokens/keys

# 4. Launch
npm start
```

<details>
<summary><b>🐳 Deploy with Docker</b></summary>

<br/>

```bash
docker build -t rage-bots .
docker run -d --env-file .env --name rage-bots --restart unless-stopped rage-bots
```

</details>

<details>
<summary><b>☁️ Keep it alive with PM2</b></summary>

<br/>

```bash
npm install -g pm2
pm2 start index.js --name rage-bots
pm2 save
pm2 startup
```

</details>

## `05` ⚙️ Configuration

Create a `.env` file in the root directory:

```env
BOT_TOKEN=your_token_here
PREFIX=!
OWNER_ID=your_id_here
```

<div align="center">

| Variable | Required | Description |
|:---|:---:|:---|
| `BOT_TOKEN` | ✅ | Your bot's authentication token |
| `PREFIX` | ❌ | Command prefix — defaults to `!` |
| `OWNER_ID` | ✅ | Your user ID for admin-only commands |

</div>

```bash

npm run dev      # run with hot-reload
```
